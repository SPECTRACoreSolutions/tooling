## Microsoft Fabric

- CLI: `fab` (installed in devcontainer).
- Key Railway vars (shared): `SPECTRA_FABRIC_CLIENT_ID`, `SPECTRA_FABRIC_CLIENT_SECRET`, `SPECTRA_FABRIC_TENANT_ID`, `SPECTRA_FABRIC_CAPACITY_ID`.
- Auth: `fab login` (uses AAD; no secrets in repo).
- Docs: https://learn.microsoft.com/fabric/
- Data engineering (incl. notebook parameters): https://learn.microsoft.com/en-us/fabric/data-engineering/
