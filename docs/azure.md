## Azure

- CLI: `az` (installed in devcontainer).
- Tenant id (Railway var): `SPECTRA_AZURE_TENANT_ID` (set to `89aaf206-0c55-4dd7-9147-9c95c1a0ff39`).
- Login: `az login` (device/browser).
- Common use: `az account show`, `az group list`, `az keyvault ...` as needed.
- Docs: https://learn.microsoft.com/cli/azure/
