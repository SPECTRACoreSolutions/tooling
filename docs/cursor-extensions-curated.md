## Quality Criteria

Extensions included must meet ONE of:
- ✅ >1M downloads on marketplace
- ✅ >1k GitHub stars
- ✅ Official Microsoft/JetBrains/HashiCorp extension
- ✅ Recommended by >3 major tech companies/communities
- ✅ Industry standard tool (e.g., Docker, Kubernetes)

**We DON'T include**:
- ❌ Niche extensions with <100k downloads
- ❌ Unmaintained (no updates in 12+ months)
- ❌ Single-developer hobby projects (unless exceptional)
- ❌ Duplicate functionality of better-established tools

## Extension Profiles (New Feature!)

VS Code now supports **extension profiles** - different sets of extensions per project:

### Create SPECTRA Profiles

**SPECTRA Data Profile**:
```json
{
  "name": "SPECTRA Data",
  "extensions": [
    "ms-python.python",
    "ms-python.vscode-pylance",
    "ms-toolsai.jupyter",
    "rangav.vscode-thunder-client",
    "mechatroner.rainbow-csv",
    "mtxr.sqltools",
    "mikestead.dotenv"
  ]
}
```

**SPECTRA Docs Profile**:
```json
{
  "name": "SPECTRA Docs",
  "extensions": [
    "yzhang.markdown-all-in-one",
    "hediet.vscode-drawio",
    "shd101wyy.markdown-preview-enhanced"
  ]
}
```

## Performance Notes

### Extensions That Impact Performance

**Heavy (>100ms startup time)**:
- GitLens (configurable, can be lightened)
- Jupyter (only enable when needed)
- Docker (only when using containers)
- Remote - SSH (only when connecting)

**Light (<50ms startup time)**:
- DotENV
- Rainbow CSV
- Prettier
- YAML

**Recommendation**: Use extension profiles to enable only what you need per project.
