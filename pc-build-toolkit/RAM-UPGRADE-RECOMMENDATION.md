## 🔧 Installation Notes

### **When You Get New RAM:**

1. **Remove old RAM** (4x 16GB sticks)
2. **Install new RAM** in slots A2 and B2 (usually 2nd and 4th slots)
3. **Boot to BIOS**
4. **Enable XMP Profile 1** (should work now!)
5. **Verify speed** in Windows (should show 7200 MHz)
6. **Run stress test** (MemTest86 or Windows Memory Diagnostic)

### **BIOS Settings:**
- Enable XMP Profile 1
- Verify speed shows 7200 MT/s (or 8000 if you go that route)
- Check timings match spec sheet
- Save and exit
