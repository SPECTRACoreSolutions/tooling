🚀 QUICK START - NEW BUILD:

1. Boot from USB (if doing fresh Windows install)
   OR
   Boot to Windows and insert USB

2. Install drivers in this order:
   a) Chipset driver (02-Drivers/MSI-MEG-Z890-GODLIKE/Chipset/)
   b) Restart
   c) LAN driver (for internet)
   d) Run Windows Update
   e) Install remaining drivers

3. Install MSI Center (03-AIO-RGB/MSI-Center/)

4. Run stress tests (04-Utilities/Stress-Testing/)
   - Prime95 for CPU stability (10-30 minutes)
   - MemTest86 for RAM (overnight recommended)

5. Benchmark and enjoy!

⚠️ TROUBLESHOOTING:

No display after build?
    → Check RAM is fully seated (clicks twice)
    → Check CPU power cable (8-pin at top left)
    → Check GPU power cables
    → Try one RAM stick in slot A2

No internet after Windows install?
    → Install LAN driver from 02-Drivers/MSI-MEG-Z890-GODLIKE/LAN/

Windows won't activate?
    → Sign in with Microsoft account
    → Run: slmgr /ato

High temps?
    → Check AIO pump is powered and running
    → Check thermal paste application
    → Check radiator fans are spinning
