🔌 PHASE 1: SAFE SHUTDOWN & REMOVAL
═══════════════════════════════════════════════════════════════════════════

1. Shut down Windows properly (Start → Shutdown)
2. Flip PSU switch to OFF (back of PC)
3. Unplug power cable from wall
4. Press power button 5-10 times (discharge capacitors)
5. Remove all external cables:
   - Monitor cables
   - USB devices
   - Network cable
   - Audio cables
   - Any other peripherals

6. Move PC to workspace with good lighting
7. Open side panel(s)
8. 📸 Take photo of cable routing (for reference later!)

⚠️  CRITICAL: Touch your PC case frequently to ground yourself!

🗑️  PHASE 2: REMOVE OLD COMPONENTS
═══════════════════════════════════════════════════════════════════════════

ORDER MATTERS! Follow this sequence:

1. DISCONNECT ALL MOTHERBOARD CABLES:
   [ ] 24-pin ATX power (squeeze clip, pull straight)
   [ ] 8-pin CPU power (top left of board)
   [ ] Front panel connectors (tiny pins at bottom)
   [ ] USB headers (front panel USB ports)
   [ ] Audio header (front panel audio)
   [ ] All fan headers
   [ ] Any RGB/LED headers
   [ ] SATA cables (if any)

2. REMOVE GPU (if installed):
   [ ] Unscrew bracket from case (back of PC)
   [ ] Press PCIe slot release tab (on motherboard)
   [ ] Pull GPU straight out (may need wiggling)
   [ ] Set aside in anti-static bag

3. REMOVE RAM:
   [ ] Push clips on both ends outward
   [ ] Pull RAM straight up
   [ ] Keep all 4 sticks together safely

4. REMOVE M.2 DRIVES:
   [ ] Locate M.2 drives (usually under heatsinks)
   [ ] Remove heatsink (if present)
   [ ] Unscrew tiny M.2 screw
   [ ] Pull drive out at 30° angle
   [ ] Keep screws in safe place!

5. REMOVE OLD AIO COOLER:
   [ ] Disconnect pump power cable
   [ ] Disconnect pump USB cable (if any)
   [ ] Disconnect ARGB cable (if any)
   [ ] Unscrew radiator from case (3-6 screws)
   [ ] Unscrew pump from CPU (4 screws, X pattern)
      ⚠️  Loosen in X pattern: a bit of each screw at a time!
   [ ] Lift cooler carefully (thermal paste will be sticky)
   [ ] Set aside

6. CLEAN OLD THERMAL PASTE:
   [ ] Put isopropyl alcohol on coffee filter
   [ ] Gently wipe CPU surface (IHS)
   [ ] Wipe until completely clean and shiny
   [ ] Let air dry (30 seconds)

7. REMOVE OLD MOTHERBOARD:
   [ ] Count motherboard screws (usually 8-9)
   [ ] Unscrew all motherboard screws
   [ ] Gently lift motherboard straight up
   [ ] Place in anti-static bag or on box

8. OPTIONAL: CLEAN CASE:
   [ ] Compressed air to blow out dust
   [ ] Wipe down case interior
   [ ] Check/clean dust filters

─────────────────────────────────────────────────────────────────────────────
STEP 1: INSTALL CPU (Core Ultra 9 285K) - 5 MINUTES
─────────────────────────────────────────────────────────────────────────────

⚠️  MOST CRITICAL STEP - GO SLOW!

[ ] Locate CPU socket (large square with metal cover)
[ ] Lift retention arm (moves to the side, click)
[ ] Remove plastic socket cover (save for warranty!)
[ ] Remove CPU from package:
    - Hold by edges ONLY
    - Never touch bottom pins/contacts
[ ] Locate orientation marker:
    - CPU has small triangle in one corner
    - Socket has matching triangle
    - MUST align correctly!
[ ] Place CPU in socket:
    - Should drop in with ZERO force
    - If it doesn't sit flat → STOP, check orientation
    - Should sit completely flat in socket
[ ] Close retention arm:
    - Will need slight pressure
    - Clicks into place when locked

✅ CPU should be completely flat, retention arm locked

─────────────────────────────────────────────────────────────────────────────
STEP 2: INSTALL RAM - 2 MINUTES
─────────────────────────────────────────────────────────────────────────────

[ ] Check Z890 manual for slot configuration
    (All 4 slots for quad-channel: A1, A2, B1, B2)
[ ] Open clips on both ends of each slot
[ ] Align notch on RAM with notch in slot
[ ] Press down firmly until BOTH clips click
    - Requires firm pressure
    - Should hear/feel two clicks
[ ] Verify: You should NOT be able to remove RAM without pressing clips

✅ All 4 RAM sticks installed, clips locked

─────────────────────────────────────────────────────────────────────────────
STEP 3: INSTALL MOTHERBOARD IN CASE - 10 MINUTES
─────────────────────────────────────────────────────────────────────────────

[ ] Install I/O shield if separate (snap into place from inside)
    (Z890 GODLIKE likely has integrated I/O shield)
[ ] Check standoffs align with motherboard holes
[ ] Lower motherboard into case carefully
[ ] Align I/O ports with I/O shield
[ ] Check all standoff screws align with holes
[ ] Screw in motherboard:
    - Start with corners (don't fully tighten yet)
    - Then middle screws
    - Finally tighten all (snug, not gorilla-tight!)

✅ Motherboard secure, doesn't flex when pressed

─────────────────────────────────────────────────────────────────────────────
STEP 4: INSTALL MSI MAG CORELIQUID i360 AIO - 20 MINUTES
─────────────────────────────────────────────────────────────────────────────

🔗 Follow MSI CoreLiquid i360 manual for LGA 1851!

GENERAL STEPS:

[ ] Install backplate behind motherboard:
    - Feed posts through holes around CPU socket
    - Secure from front with standoffs

[ ] Check thermal paste:
    - Pump should have pre-applied paste
    - If not: apply pea-sized dot to CPU center

[ ] Mount pump block to CPU:
    - Align with backplate posts
    - Start all 4 screws loosely
    - Tighten in X pattern: a bit of each at a time
    - Final tightness: snug but not over-tight
    - Orient pump display how you want (rotatable)

[ ] Install radiator in case:
    OPTION A: Top mount (exhaust) - Quieter, slightly warmer CPU
    OPTION B: Front mount (intake) - Cooler CPU, slightly warmer system
    
    💡 Recommendation: Top mount as exhaust
    
    - Tubes should route naturally (no kinking)
    - Tubes at bottom of radiator (helps with air bubbles)
    - Screw radiator + fans to case

[ ] Connect cables:
    - Pump power → CPU_FAN or PUMP_FAN header on motherboard
    - Radiator fans → SYS_FAN headers or fan hub
    - ARGB cable → JRAINBOW header (3-pin 5V ARGB)
    - USB cable → USB 2.0 header (for pump display control)

✅ AIO installed, all cables connected, pump feels firm

─────────────────────────────────────────────────────────────────────────────

─────────────────────────────────────────────────────────────────────────────

[ ] Push down flat
[ ] Screw in with tiny M.2 screw
[ ] Replace heatsink
[ ] Repeat for all M.2 drives

GPU (if you have one):
[ ] Remove PCIe slot covers from case
[ ] Align GPU with top PCIe x16 slot
[ ] Push down firmly until click
[ ] Screw bracket to case
[ ] Connect PCIe power cables to GPU (if needed)

✅ All storage drives installed
✅ GPU installed and secured

─────────────────────────────────────────────────────────────────────────────
STEP 6: CONNECT ALL CABLES - 15 MINUTES
─────────────────────────────────────────────────────────────────────────────

POWER CABLES:
[ ] 24-pin ATX to motherboard (main power, right side)
    - Push until CLICK!
[ ] 8-pin CPU power to top left of motherboard
    - May be 2x 4-pin or 1x 8-pin
[ ] PCIe power to GPU (if needed)
    - 8-pin or 6+2 pin connectors

FRONT PANEL CONNECTORS (consult Z890 manual!):
[ ] PWR_SW (power button) - CRITICAL!
[ ] RST_SW (reset button)
[ ] PWR_LED + and -
[ ] HDD_LED

💡 These tiny pins are labeled. Match labels on motherboard.
   PWR_SW is the most critical - PC won't start without it!

HEADERS:
[ ] USB 3.0 header (blue, 19-pin)
[ ] USB 2.0 header(s) (black, 9-pin)
[ ] Audio header (HD_AUDIO)
[ ] All remaining fan headers

SATA (if you have SATA drives):
[ ] SATA data cables to motherboard
[ ] SATA power from PSU to drives

✅ All cables connected
✅ Nothing loose or unplugged

═══════════════════════════════════════════════════════════════════════════
⚠️  TROUBLESHOOTING: NO DISPLAY
═══════════════════════════════════════════════════════════════════════════

If fans spin but no display:

1. CHECK RAM:
   - Reseat all RAM sticks (remove and reinstall)
   - Try one stick in slot A2
   - Try each stick individually

2. CHECK CABLES:
   - Monitor cable connected to GPU (not motherboard)?
   - Both power cables connected (24-pin + 8-pin)?

3. CHECK CPU:
   - Was it oriented correctly?
   - Did it sit flat?
   - Retention arm locked?

4. CHECK DEBUG LEDS:
   - Z890 GODLIKE has debug LED display
   - Note the code, look up in manual

5. CLEAR CMOS:
   - Look for CMOS clear button on motherboard
   - Or remove CMOS battery for 30 seconds
