# Cool Coding Features for Stream Deck

**Advanced and creative ways to use Stream Deck for coding**

---

## 🚀 Top 10 Cool Features

### **1. Multi-Action Git Workflows**

**"Quick Commit & Push" (One Button):**
1. Opens terminal
2. Types: `git add .`
3. Presses Enter
4. Waits 1 second
5. Types: `git commit -m "WIP"`
6. Presses Enter
7. Waits 1 second
8. Types: `git push`
9. Presses Enter

**Result:** One button press = full commit workflow!

---

### **2. Project Quick Switch**

**Create buttons for each project:**
- **Notifications** → Opens `Core/notifications` in Cursor
- **Assistant** → Opens `Core/assistant` in Cursor
- **Graph** → Opens `Core/graph` in Cursor

**Or use a Folder:**
- **SPECTRA Projects** (folder)
  - Inside: All project buttons
  - Back button to return

**Result:** Switch projects instantly!

---

### **3. Code Snippets**

**Quick code insertion:**

**"Python Function":**
```python
def function_name():
    """Docstring here."""
    pass
```

**"Test Function":**
```python
def test_function_name():
    """Test description."""
    assert True
```

**"SPECTRA Service Template":**
- Full service boilerplate
- Includes imports, structure, comments

**Result:** No more typing boilerplate!

---

### **4. Terminal Command Shortcuts**

**One-button commands:**

- **"Run Tests"** → Types `pytest` + Enter
- **"Lint Code"** → Types `ruff check .` + Enter
- **"Sync Index"** → Runs cosmic index script
- **"Git Status"** → Types `git status` + Enter

**Result:** Common commands at your fingertips!

---

### **5. File Navigation**

**Quick access to important files:**

- **"Labs Queue"** → Opens `ideas.json`
- **"Service Catalog"** → Opens `service-catalog.yaml`
- **"Standards"** → Opens `Core/standards` folder
- **"Journal"** → Opens `Core/memory/journal` folder
- **"Documentation"** → Opens `Core/docs` folder

**Result:** Never hunt for files again!

---

### **6. Folder Organisation**

**Nested button groups:**

**"Git Operations" (Folder):**
- Status, Commit, Push, Pull, Branch, Stash, Log
- Back button to return

**"SPECTRA Services" (Folder):**
- Notifications, Assistant, Graph, Portal, etc.
- Back button to return

**"Quick Commands" (Folder):**
- Test, Lint, Sync, Status, Docs
- Back button to return

**Result:** Organised, not cluttered!

---

### **7. Complex Workflows**

**"Start New Feature" (Multi-Action):**
1. Open Cursor
2. Wait 2 seconds
3. Open terminal
4. Type: `git checkout -b feature/new-feature`
5. Press Enter
6. Open Labs queue
7. Type: `git status`

**"Complete Feature" (Multi-Action):**
1. Run tests
2. Wait for completion
3. Lint code
4. Wait for completion
5. Git status
6. Open commit template

**Result:** Automate entire workflows!

---

### **8. Dynamic Content (With Plugins)**

**"Git Branch Display":**
- Shows current Git branch
- Updates automatically
- Changes colour if branch is dirty

**"Service Status":**
- Shows Railway service status
- Green = running, Red = down
- Updates every few minutes

**"Time Tracker":**
- Shows current session time
- Start/stop timer
- Log time to file

**Result:** Live information at a glance!

---

### **9. Debugging Shortcuts**

**Development tools:**

- **"Toggle Breakpoint"** → F9
- **"Start Debugging"** → F5
- **"Step Over"** → F10
- **"Step Into"** → F11
- **"Restart Debugger"** → Ctrl+Shift+F5

**Result:** Debugging without keyboard!

---

### **10. Environment Switching**

**Context switching:**

**"Switch to Production":**
- Sets environment variable
- Opens production dashboard
- Changes terminal prompt

**"Switch to Development":**
- Sets environment variable
- Opens dev tools
- Changes terminal prompt

**Result:** Switch contexts instantly!
