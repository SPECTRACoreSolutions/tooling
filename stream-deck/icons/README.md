# Stream Deck Icons

**Custom icons for Stream Deck buttons**

---

## 📋 Icon Guidelines

### **Size Requirements**

- **Recommended:** 72x72 pixels
- **Minimum:** 72x72 pixels
- **Maximum:** 288x288 pixels (will be scaled down)

### **Format**

- **PNG** (recommended - supports transparency)
- **JPG** (acceptable - no transparency)
- **SVG** (if supported by Stream Deck version)

### **Design Tips**

- Use high contrast for visibility
- Keep designs simple (small screen size)
- Use SPECTRA branding where appropriate
- Maintain consistent style across icons

## 🎨 Creating Custom Icons

### **Using Image Editor**

1. Create 72x72 pixel canvas
2. Design icon (simple, high contrast)
3. Export as PNG
4. Save to appropriate subdirectory
5. Use in Stream Deck software

### **Using Online Tools**

- [Stream Deck Icon Generator](https://www.elgato.com/en/stream-deck-icon-generator)
- [Favicon Generator](https://favicon.io/) (resize to 72x72)
- [Canva](https://www.canva.com/) (custom size)

---

## 📝 Icon Naming Convention

Use descriptive names:

- `spectra-logo.png`
- `cursor-icon.png`
- `git-icon.png`
- `deploy-icon.png`
- `service-icon.png`

---

## 🔄 Adding Icons

1. Create or download icon (72x72 PNG)
2. Save to appropriate directory
3. Use in Stream Deck software:
   - Click button
   - Click icon area
   - Choose File
   - Select icon
